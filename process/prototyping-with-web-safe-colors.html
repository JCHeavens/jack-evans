Prototyping with Web Safe Colors

26 June 2015

These days, you can use any color you like on the web but should you? When I prototype a design, I start with old school web safe color schemes.

For me, it’s an easy way to set some contrast and hierarchy in the interface early on in the design process, especially with links, navigation and layout.

You may end up diverging from the web safe colors you started with, but it’s useful as an approximation process and it results in color schemes that are more concise, meaningful and visually effective.
